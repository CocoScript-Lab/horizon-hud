# HORIZON HUD

Forza Horizon 6（FH6）の**公式テレメトリ（Data Out）**を受信し、ゲームとは別の画面に
速度・回転数・ギア・ブースト・Gフォースなどを表示する、**完全ローカル・読み取り専用**の
ダッシュボード（HUD）アプリです。SimHub などの外部ツールは不要、外部通信もしません。

> ⚠️ これは非公式のファン制作ツールです。Microsoft / Playground Games とは一切関係が
> ありません。"Forza Horizon" は Microsoft の商標です。ゲームから抽出した画像などの
> 著作物は同梱しておらず、表示するデータは Forza 公式の Data Out（数値）のみです。

## ダウンロード

このページ右側の **[Releases](../../releases/latest)** から最新版の
`HORIZON-HUD-vX.Y.Z.zip` をダウンロードしてください。

| 項目 | 要件 |
| --- | --- |
| OS | Windows 11（64bit） |
| ゲーム | Forza Horizon 6（Data Out 対応） |

## インストールと起動

1. ダウンロードした `HORIZON-HUD-vX.Y.Z.zip` を展開し、中の `HORIZON-HUD-Setup.exe` を実行。
2. 初回起動時に「**WindowsによってPCが保護されました**」と表示された場合は、**「詳細情報」→「実行」** で進めてください。
3. インストール後、デスクトップの「**HORIZON HUD**」アイコンから起動。

## ゲーム側の設定（重要）

FH6 の 設定 → HUD/ゲームプレイ → **Data Out を ON** にし、同じPCで動かす場合は

- IP：`127.0.0.1`
- ポート：`20440`

を指定してください。運転を始めると HUD に数値が表示されます。

## スマホ・タブレットで見る

1. アプリの「**接続**」タブで「スマホ・タブレットに表示する」を ON。
2. 表示された **QRコード**をスマホで読み取る（Chrome 推奨）。
3. 同じ Wi-Fi なら、スマホにHUDが表示されます（表示専用）。
   - 初回は Windows ファイアウォールで「**プライベート ネットワーク**」を許可してください。

## 安全性（読み取り専用）

- ゲームメモリの読み書き・改変、入力の自動操作、通信の傍受は**一切行いません**。
- 車両情報は FH6 公式の Data Out からのみ取得します。
- 外部インターネットへは通信しません（LAN公開時も同一Wi-Fi内のみ）。
